# ChargePack
充電関連のスクリプト詰め合わせです

## passive
- ChargeLightning
    麻痺を付与されたとき、麻痺の値だけ充電を得て麻痺を消去する
- ChargeEfficient
    充電を得たとき、追加で1得る
- ChargeHeal
    攻撃的中時、自分に充電があるなら体力を5回復。充電を1消費する
- ChargeBullet
    遠距離ダイスの使用時、充電を2消費する。充電を消費したならダイスの威力+2、ダメージ量+3
- ChargeBattery
    舞台開始時、充電を6得る
- ChargeElecy
    攻撃的中時、充電が11以上なら麻痺1を付与する
- ChargeShock
    攻撃を受けた時、相手に麻痺1を付与し充電2を得る
